Кислов Влад / ЕПИМ 2-23 / Курсовая работа / Проектирование распределенных информационных систем 

Программа эмулирует работу распределённого протокола консенсуса Paxos и. 
Функции:
	*Запускать несколько узлов (acceptors и proposers) на одной машине.
	*Передавать сообщения между ними по UDP.
	*Обрабатывать сообщения согласно алгоритму Paxos.
	*Отслеживать состояние каждого узла во времени.
	*Визуализировать поведение системы в виде графического интерфейса с фигурами и стрелками.

Данный проект позволяет:
	*Увидеть и понять, как работает Paxos на практике.
	*Наблюдать за процессом достижения согласия между независимыми агентами.
	*Проверить устойчивость системы к сбоям.
	*Проанализировать хронологию сообщений.

Что отображается в интерфейсе:
	*Фигуры на холсте (Canvas): Круги — Acceptor. Квадраты — Proposer.
	*Стрелки между узлами:
		Бордовые — сообщения от Proposer.
		Синие — сообщения от Acceptor.
		Толщина стрелки — показывает, насколько недавно было взаимодействие.

Текстовая область:
	*Показывает текущее состояние каждого узла (номер, эпоха, последний тип сообщения и др.).
	*Отображает технические детали: текущая эпоха, ID сообщений, полученные/отправленные значения.
